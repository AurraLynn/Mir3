# 梦想传奇3游戏网站

这是《梦想传奇3》游戏的官方网站，采用现代化的设计和技术栈，为玩家提供游戏信息、下载和社区互动等功能。

## 项目结构

```
mir3/
├── index.html      # 网站主页
├── styles.css      # 自定义样式
├── news.js         # 新闻功能模块
├── screenshot-viewer.js  # 截图查看器模块
├── download.js     # 下载功能模块
├── main.js         # 主要功能模块
└── README.md       # 项目说明
```

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome 图标库
- GSAP 动画库
- Particles.js 粒子背景

### 1. 响应式设计
- 适配桌面、平板和移动设备
- 移动端专用菜单

### 2. 视觉效果
- 全屏视频背景
- 粒子动画效果
- 滚动渐显动画

### 3. 游戏内容展示
- 游戏特色介绍

### 4. 新闻公告系统
- 新闻列表展示

### 5. 下载系统

- 平滑滚动导航

- 截图查看器

## 使用说明

1. 打开 `index.html` 文件即可访问网站
2. 网站支持以下交互功能：
   - 导航菜单点击跳转到对应区域
   - 新闻卡片点击查看详情
   - 游戏截图点击放大查看
   - 下载按钮触发下载流程
   - 返回顶部按钮快速回到页面顶部

此版本为《梦想传奇3》游戏网站的完整版本，包含所有原始文案，特别是游戏背景故事部分的详细内容。网站采用现代化的设计和交互效果，为玩家提供沉浸式的浏览体验。
